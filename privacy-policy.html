⚔️ Privacy Policy

Quest Board RPG  ·  Last updated: June 2026

Quest Board RPG ("the App") is a free, browser-based productivity application developed by JVDesignStudio. This policy explains how data is handled when you use the App.

1. Data We Collect

We collect no personal data. Quest Board RPG does not have a server, a database, or any backend infrastructure. There is no account system and no registration required.

2. Data Stored on Your Device

The App stores data exclusively on your own device using the browser's localStorage API. This includes:

  • Your quest cards, columns, and board state
  • Your RPG character stats, level, XP, and gold
  • Your notes, habits, and self-care records
  • Your daily check-in responses and mood data
  • Your app preferences (theme, class, player name)

This data never leaves your device unless you explicitly use the GitHub Gist Sync feature (see Section 4).

3. Analytics

The Quest Board landing page (quest-board-page.html) uses Google Analytics (GA4) to collect anonymised page view data such as visit counts and device type. The Quest Board application itself (quest-board.html / index.html) does not use analytics.

Google Analytics data is processed in accordance with Google's own privacy policy. You can opt out using the Google Analytics Opt-out Browser Add-on.

4. GitHub Gist Sync (Optional Feature)

The App includes an optional cloud backup feature that, if you choose to enable it, sends your board data to GitHub Gist using a Personal Access Token you provide. This data is:

  • Sent directly from your browser to GitHub's API
  • Stored in a private Gist accessible only with your GitHub credentials
  • Never seen or processed by JVDesignStudio

Your GitHub token is stored only in your browser's localStorage and is transmitted only to api.github.com. This feature is entirely opt-in and can be disabled at any time by not entering a token.

5. Calendar Import (Optional Feature)

The App allows you to import .ics calendar files. These files are processed entirely within your browser — no calendar data is uploaded to any server. The imported events are stored in your device's localStorage alongside your other board data.

6. Third-Party Services

  • Google Fonts — used for typography. Google may log font requests.
  • Google Analytics — used on the landing page only (see Section 3).
  • GitHub API — used only if you enable Gist sync (see Section 4).

7. Children's Privacy

Quest Board RPG does not knowingly collect any information from children under 13. Since the App stores no personal data on any server, it presents minimal privacy risk for users of any age.

8. Data Deletion

To delete all your Quest Board data, clear your browser's localStorage for the domain where the App is hosted. In most browsers: Settings → Privacy → Clear browsing data → Cached data and Cookies. Alternatively, opening developer tools (F12), navigating to Application → Local Storage, and deleting all keys beginning with qb- will remove all App data.

9. Changes to This Policy

If this policy is updated, the "Last updated" date at the top will change. Continued use of the App constitutes acceptance of any changes.

10. Contact
